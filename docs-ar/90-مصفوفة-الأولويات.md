# مصفوفة الأولويات

كل ميزة في المجموعة، مع أولويتها كما وردت في مواصفة وحدتها ومبرّر من سطر واحد مرتبط بأهداف المنصة (الوثيقة 00). الأولويات تجيب عن سؤال واحد: *إلى أي مدى تقف هذه الميزة مباشرةً بين شخص وأذى، وما الذي يجب أن يوجد قبل أن تعمل؟*

## حرجة (Critical) — سبب وجود المنصة؛ تُطلق أولًا وإلا فلا معنى لما عداها

| الميزة | المواصفة | المبرّر |
|---|---|---|
| أرقام الطوارئ | [21](modules/21-مركز-المعلومات.md) | نقرتان إلى مساعدة محترفة؛ تعمل حين ينهار كل شيء آخر؛ تافهة البناء |
| خريطة الحرائق المباشرة | [10](modules/10-خريطة-الحرائق-المباشرة.md) | الصورة التشغيلية المشتركة التي يتعلّق بها كل طرف معنيّ وكل وحدة |
| الإبلاغ عن الحوادث | [20](modules/20-الإبلاغ-عن-الحوادث.md) | التغذية الخام للخريطة؛ الكشف المبكّر أرخص إطفاء ممكن |
| تنبيهات الطوارئ | [11](modules/11-تنبيهات-الطوارئ.md) | أكثر دفعة منقذة للأرواح مباشرةً: أخلِ الآن، وإلى هنا، وتجنّب ذلك الطريق |
| طلبات الاستغاثة | [12](modules/12-طلبات-الاستغاثة.md) | تحويل "من يحتاج المساعدة" من مكالمات ضائعة إلى طابور مفروز |
| إغلاق الطرق | [19](modules/19-الطرق-والمسارات-الآمنة.md) | المعلومة التي يقتل غيابها بأكثر صورة مباشرة أثناء الإخلاء |
| دليل المآوي (عمومي) | [15](modules/15-إدارة-المآوي.md) | التنبيهات بلا جواب عملي عن "إلى أين نذهب" هي نصف إخلاء |
| المفقودون | [13](modules/13-المفقودون-ولم-شمل-العائلات.md) | أزمة ما بعد الإخلاء؛ تحلّ محلّ بحث فيسبوك الأعمى تجاه الخصوصية |

## عالية (High) — تجعل النواة الحرجة منسَّقة وآمنة وصادقة؛ تُطلق في الموسم نفسه

| الميزة | المواصفة | المبرّر |
|---|---|---|
| تسجيل "أنا بخير" *(جديدة)* | [13](modules/13-المفقودون-ولم-شمل-العائلات.md) | أرخص مخفِّف حِمل: يمنع ملفات المفقودين وطوفان مكالمات الاطمئنان |
| لمّ شمل العائلات | [13](modules/13-المفقودون-ولم-شمل-العائلات.md) | المطابِق الذي يحوّل قوائم التسجيل وبلاغات المفقودين إلى أشخاص وُجدوا |
| إدارة المتطوعين | [14](modules/14-إدارة-المتطوعين.md) | من يستطيع المساعدة، وبماذا، متحقَّقًا منه — السجلّ الذي تقوم عليه المهام |
| مهام المتطوعين | [14](modules/14-إدارة-المتطوعين.md) | تحوّل الرغبة إلى عمل مزوَّد بطاقم، محدَّد المكان والزمن، وآمن |
| تسجيل سلامة المتطوّع ونداء الاستغاثة *(جديدة)* | [14](modules/14-إدارة-المتطوعين.md) | رفيق غير قابل للتفاوض للمهام؛ المتطوعون يعودون |
| تشغيل المآوي | [15](modules/15-إدارة-المآوي.md) | يجعل بيانات المآوي ذاتية الاستدامة؛ حيث تلتقي خمس وحدات |
| إدارة الموارد | [16](modules/16-الموارد-والتوزيع.md) | إشارة مخزون خشنة لكن مصانة؛ تتيح للمنسّقين الدفع قبل النقص |
| طلبات التموين (لوحة الاحتياجات) | [16](modules/16-الموارد-والتوزيع.md) | قفل المطالبة يقتل الطلبات المكرّرة والثقوب السوداء؛ قلب اللوجستيك |
| توزيع الماء | [16](modules/16-الموارد-والتوزيع.md) | أول مؤونة للبقاء حين تحترق الشبكات؛ جداول يستطيع الناس اللحاق بها |
| توزيع الغذاء | [16](modules/16-الموارد-والتوزيع.md) | أكثر المساعدات تكرارًا *وأكثرها* فجوات؛ وعرض التغطية يصلح الأمرين |
| التبرعات العينية | [17](modules/17-التبرعات.md) | توجّه الكرم القائم نحو حاجات حقيقية؛ و"غير مطلوب" نصف القيمة |
| التنسيق الطبي | [18](modules/18-التنسيق-الطبي.md) | يوجّه الحروق إلى الباب الصحيح؛ ويُطفي موجة الرعاية المزمنة الصامتة على السطح |
| المسارات الآمنة | [19](modules/19-الطرق-والمسارات-الآمنة.md) | الحلقة الأخيرة في سلسلة الإخلاء؛ بخطوة بعد بيانات الإغلاق عن قصد |
| تحديثات المجتمع المحلي | [21](modules/21-مركز-المعلومات.md) | آلة إزاحة الشائعات: حقائق محلية متحقَّق منها، للقراءة فقط |
| المعلومات كثيرة الطلب | [21](modules/21-مركز-المعلومات.md) | أجوبة مخزَّنة مسبقًا ومسموعة عن الأسئلة التي يرتجلها الذعر ارتجالًا سيّئًا |
| مراكز الطوارئ المحلية | [21](modules/21-مركز-المعلومات.md) | كل طريق رقمي مسدود يهبط عند مكتب مادي فيه إنسان |
| الإبلاغ عن التضليل والشارات *(جديدة)* | [30](30-مقترحات-ميزات-جديدة.md) | يُبقي سُلّم التحقّق صادقًا على مقياس المواطنين؛ ويصل إلى داخل واتساب |

## متوسطة (Medium) — قيمة حقيقية، لكنها تستهلك بيانات أو وقت هدوء يجب أن تنتجه الطبقات الأسبق

| الميزة | المواصفة | المبرّر |
|---|---|---|
| توزيع الوقود | [16](modules/16-الموارد-والتوزيع.md) | حيوي لكن ضيّق؛ محكوم عمدًا ببوابة المنسّق؛ رخيص على قضبان لوحة الاحتياجات |
| دليل التبرعات المالية | [17](modules/17-التبرعات.md) | قيمة ثقة عالية، لكن توقيتها ليس حيويًا؛ يحتاج نضج الاعتماد |
| لوحة الإحصائيات العامة | [22](modules/22-لوحة-الإحصائيات-العامة.md) | الشفافية تحتاج أولًا أرقامًا تستحقّ العرض |
| تقارير الأضرار | [23](modules/23-الأضرار-والتعافي.md) | بعد الأزمة بطبيعتها — لكن يجب أن تكون *جاهزة* قبل انتهاء الموسم الأول |
| متابعة التعافي | [23](modules/23-الأضرار-والتعافي.md) | لوحة مساءلة الذيل الطويل؛ تتبع سجلّ الأضرار |
| سجلّ الأشخاص الهشّين *(جديد)* | [30](30-مقترحات-ميزات-جديدة.md) | مردود من الدرجة الحرجة يوم الحريق؛ يُبنى في وقت الهدوء فلا يحجز النسخة الأولية |
| مسار الماشية والفلاحة *(جديد)* | [30](30-مقترحات-ميزات-جديدة.md) | نوع الطلب والأدلّة يركبان المضيفات بكلفة تقارب الصفر؛ طبقات المساعدة هي الجزء المتوسط |
| قادة الإبلاغ *(جديد)* | [30](30-مقترحات-ميزات-جديدة.md) | الكيلومتر الأخير البشري؛ رخيص تنظيميًا، ويتوسّع مع أصعب المستخدمين |
| وضع التحضير لما قبل الموسم *(جديد)* | [30](30-مقترحات-ميزات-جديدة.md) | فرق الموسم الثاني بين منصّة تتحلّل وأخرى تتراكم |
