# نجدة (NAJDA) — وثائق تصميم المنصة

**نجدة** منصة وطنية لتنسيق الاستجابة لحرائق الغابات في الجزائر. تربط المواطنين ورجال الإنقاذ والمتطوعين والجمعيات (NGOs) حول صورة واحدة مشتركة للأزمة: أين تشتعل الحرائق، ومن يحتاج إلى المساعدة، وإلى أين يذهب الناس، وما هو المطلوب.

هذه الوثائق **مواصفة منتج وتجربة استخدام (product & UX specification)**. تصف الميزات والمستخدمين والتفاعلات ومسارات المستخدم الكاملة والأولويات. ولا تتضمّن — عن قصد — أي بنية تقنية.

> هذه ترجمة عربية لمجموعة الوثائق الإنجليزية الموجودة في `docs/`. النسخة الإنجليزية هي المرجع الأصلي؛ عند أي اختلاف يُرجع إليها.

## كيف تُقرأ هذه المجموعة

1. ابدأ بـ [00-الرؤية-والمبادئ.md](00-الرؤية-والمبادئ.md) — الأهداف وقواعد التصميم التي تلتزم بها كل ميزة.
2. ثم [01-المستخدمون-والأدوار.md](01-المستخدمون-والأدوار.md) — من هم المستخدمون وما الذي يستطيع كل دور فعله. كل مسارات الوحدات تستعمل أسماء الأدوار هذه.
3. كل ملف في [modules/](modules/) مواصفة قائمة بذاتها. وكل ميزة بداخله تتبع القالب نفسه: **ما هي · لماذا تهمّ · من يستخدمها · كيف يتفاعل المستخدم · مسار المستخدم · الأولوية**.
4. [90-مصفوفة-الأولويات.md](90-مصفوفة-الأولويات.md) يجمع أولوية كل ميزة في جدول واحد؛ و[91-خارطة-طريق-النسخة-الأولية.md](91-خارطة-طريق-النسخة-الأولية.md) يحوّلها إلى إطلاق على مراحل.
